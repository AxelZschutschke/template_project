5 # Overview
6 
7 CppCheck is a static code analysis tool for C/C++ code, providing support by automating parts of the code review
8 process. It is developed under GNU general public license V3.0.
9 
10 ## Get CppCheck
11 
12 Get it from your OS's package store, most Linux distributions have CppCheck available, e.g. via `apt install cppcheck`.
13 
14 Alternatively, the tool can be build manually from its [repository](https://github.com/danmar/cppcheck.git), following
15 the build instructions.
16 
17 ## Using CppCheck
18 
19 CppCheck works best beeing run on cpp (implementation) files, and has some issues with header only projects. Here, test
20 code can help to include the source code into the analysis.
21 
22 CppCheck allows for entering a suppressions file via `--suppressions-list=<file>`. The `--inconclusive` switch is useful
23 to get reports even if cppcheck has an inconclusive result. It leads to more but also to false-positive reports.
24 
25 With
26 ~~~~~.cmake
27 option( CPPCHECK "Enable cppcheck" OFF)
28 ...
29 if( CPPCHECK )
30  find_program( CPPCHECK_EXECUTABLE NAMES cppcheck )
31 
32  if( CPPCHECK_EXECUTABLE )
33  file(GLOB CPPCHECK_IN
34  ${CMAKE_CURRENT_SOURCE_DIR}/include/*
35  ${CMAKE_CURRENT_SOURCE_DIR}/src/*
36  ${CMAKE_CURRENT_SOURCE_DIR}/test/unit/*
37  ${CMAKE_CURRENT_SOURCE_DIR}/test/api/*
38  )
39  set(CPPCHECK_INCLUDE_DIRS
40  -I${CMAKE_CURRENT_SOURCE_DIR}
41  -I${CMAKE_CURRENT_SOURCE_DIR}/include
42  -I${CMAKE_CURRENT_SOURCE_DIR}/vendor
43  -I${CMAKE_CURRENT_SOURCE_DIR}/test/api/*
44  )
45 
46  set(CPPCHECK_OPTIONS
47  --xml
48  --language=c++
49  --std=c++14
50  --enable=all
51  --inconclusive )
52  set(CPPCHECK_OUT
53  --output-file=${CMAKE_CURRENT_BINARY_DIR}/cppcheck.xml )
54 
55  add_custom_target( cppcheck ALL
56  COMMAND ${CPPCHECK_EXECUTABLE}
57  ${CPPCHECK_OUT}
58  ${CPPCHECK_OPTIONS}
59  ${CPPCHECK_INCLUDE_DIRS}
60  ${CPPCHECK_IN}
61  COMMENT "Running cppcheck"
62  VERBATIM )
63  else()
64  message( "cppcheck need to be installed to generate the cppcheck report" )
65  endif()
66 endif()
67 ~~~~~
