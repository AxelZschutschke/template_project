5 # Overview
6 
7 GTest and GMock are a c/c++ testing and mocking framework respectively, actively developed and maintained by Google
8 and released under a BSD-3 clause open source license.
9 
10 Formerly both project have been developed separately, but have been merged to *googletest* now. GTest can be used
11 in a header-only manner, while GMock needs some runtime libraries to be linked against the test executables.
12 
13 ## Get GoogleTest
14 
15 Get it from your OS's package store, most Linux distributions have GoogleTest available, e.g. via `apt install
16 googletest`.
17 
18 One may get GoogleTest as source distribution as well as [git repository](https://github.com/google/googletest),
19 follow the build instructions of the libraries. Here, GoogleTest is downloaded, build and used during project
20 generation automatically, using the CMake ExternalProject module. See the main *CMakeLists.txt* file:
21 
22 ~~~~~.cmake
23 option( COMPILE_TESTS "Enable tests" ON)
24 
25 include(ExternalProject)
26 
27 ...
28 if (COMPILE_TESTS)
29  ...
30  ExternalProject_Add( googletest
31  GIT_REPOSITORY https://github.com/google/googletest.git
32  PREFIX ${CMAKE_CURRENT_BINARY_DIR}/gtest
33  INSTALL_DIR ${CMAKE_CURRENT_BINARY_DIR}
34  CMAKE_ARGS
35  -DCMAKE_INSTALL_PREFIX=${CMAKE_CURRENT_BINARY_DIR}/gtest
36  -DBUILD_SHARED_LIBS=ON
37  )
38 ~~~~~

41 ## Set up CMake Project
42 
43 For using GoogleTest from a CMake project, test cases / test runner executables need to be defined and backed with
44 source code. Each of the tests is build into and executable and linked against gtest, gmock and other relevant
45 libraries (also the projects target libraries if applicable).
46 
47 When adding the test to ctest via `add_test()`, the tests will be executed and evaluated automatically with a ctest run.
48 Add a command line option for specifying an test report format, e.g. `--gtest_output="xml:file.xml"`.
49 
50 ~~~~~.cmake
51 option( COMPILE_TESTS "Enable tests" ON)
52 
53 ...
54 if (COMPILE_TESTS)
55  file( GLOB GTEST_FILES test/unit/*)
56  foreach( FILENAME ${GTEST_FILES} )
57  get_filename_component( TESTNAME ${FILENAME} NAME_WE )
58  link_directories( ${CMAKE_CURRENT_BINARY_DIR}/gtest/lib )
59  add_executable( ${TESTNAME} ${FILENAME} )
60  add_dependencies( ${TESTNAME} googletest )
61  target_include_directories( ${TESTNAME}
62  PRIVATE
63  ${CMAKE_CURRENT_BINARY_DIR}/gtest/include
64  )
65  target_link_libraries( ${TESTNAME}
66  gtest
67  gmock )
68  add_test( NAME ${TESTNAME}
69  COMMAND ${TESTNAME} --gtest_output="xml:${TESTPATH}_gtest.xml")
70  endforeach()
71 endif ()
72 ~~~~~
